# Touchlab Limited

Welcome to the Touchlab Limited Github.  Below you can find links to our public repositories.

[Touchlab Limited Website](https://www.touchlab.io)

## Touchlab Sensors

[Touchlab Driver Ros](https://github.com/touchlab-avatarx/touchlab_driver_ros) - A C++ driver for ROS1 and ROS2

[Touchlab Comm Py](https://github.com/touchlab-avatarx/touchlab_comm_py) - A Python wrapper for the C++ driver in ROS1 and ROS2

[Touchlab Comm Py (PIP)](https://pypi.org/project/touchlab-comm-py/) -  A Python wrapper for the C++ driver in ROS1 and ROS2 (PIP installable)

[Touchlab Msgs](https://github.com/touchlab-avatarx/touchlab_msgs) - A message package for ROS1 and ROS2

[Touchlab Description](https://github.com/touchlab-avatarx/touchlab_description) - A description package for ROS1 and ROs2

[Touchlab Driver](https://github.com/touchlab-avatarx/touchlab_driver_cpp) - C++ API

## Allegro Hand

[Allegro Hand API](https://github.com/touchlab-avatarx/allegro_api) - An API for the Allegro hand in ROS1 and ROS2

[Allegro Hand Description](https://github.com/touchlab-avatarx/allegro_hand_description) - A description package for the Allegro hand in ROS1 and ROS2

[Allegro Robot Driver](https://github.com/touchlab-avatarx/allegro_robot_driver) - A robot driver for the Allegro hand in ROS1 and ROS2

## OnRobot 2FG7

[OnRobot 2FG7 API](https://github.com/touchlab-avatarx/onrobot_api) - An API for the OnRobot 2FG7 in ROS1 and ROS2

[OnRobot 2FG7 Description](https://github.com/touchlab-avatarx/onrobot_2fg7_description) - A description package for the OnRobot 2FG7 in ROS1 and ROS2

[OnRobot 2FG7 Robot Driver](https://github.com/touchlab-avatarx/onrobot_robot_driver) - A robot driver for the OnRobot 2FG7 in ROS1 and ROS2

## Schunk SVH

[Schunk SVH API](https://github.com/touchlab-avatarx/schunk_svh_api) - An API for the Schunk SVH in ROS1

[Schunk SVH Hardware Interface](https://github.com/touchlab-avatarx/schunk_svh_hardware_interface) - A Hardware interface for the Schunk SVH in ROS1

[Schunk SVH Description](https://github.com/touchlab-avatarx/schunk_svh_description) - A description package for the OnRobot 2FG7 in ROS1

[Schunk SVH Robot Driver](https://github.com/touchlab-avatarx/schunk_svh_robot_driver) - A robot driver for the Schunk SVH in ROS1
